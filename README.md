# Daily Mart — Frontend Demo

Grocery store demo for **Daily Mart**, Basmat (Owner: Suhas Karle).

## Setup
```bash
npm install
npm run dev
```

## Stack
React + Vite + Tailwind CSS + React Router + Framer Motion + Lucide React.

## Notes
- Frontend-only demo — no backend, auth, database, or payment gateway.
- WhatsApp order placement uses `wa.me` links to +91 86687 81633.
- Product images are real photos sourced from Unsplash (free to use).
- English ⇄ Marathi toggle covers common UI labels (demo only).
